# Task 4: Basic Weather App

## Description
In this task, I have created a weather app that fetches and displays current weather data for a user-specified city using a weather API. It shows temperature, humidity, and weather conditions.

## Features
- Get weather by city name
- Display current temperature
- Display humidity percentage
- Display weather conditions (Sunny, Rainy, etc.)
- Error handling for invalid city names
- API integration with OpenWeatherMap

## Technologies Used
- Python 3
- requests module (for API calls)
- JSON parsing
- OpenWeatherMap API
